*forge.txt*   Plugin documentation for forge.nvim

INTRODUCTION                            *forge-intro*

This is a fork of mmsaki/forge.nvim (https://github.com/mmsaki/forge.nvim).

forge.nvim provides commands to run Foundry:
:Forge, :Cast, :Chisel, :Anvil

COMMANDS                                *forge-commands*

:Forge [args]      Run forge commands
:Cast [args]       Run cast commands
:Chisel [args]     Run chisel scripts
:Anvil             Persistent terminal for Anvil
:AnvilKill         Kill the running Anvil instance
:ForgeBuild [args] Run forge build
:ForgeTest [args]  Run forge test
:ForgeTestMatch [name]
                   Run forge test --mt <name>; prompts for the
                   test function name when called without one
:ForgeTestSelection
                   Visual mode: run forge test --mt on the
                   selected word
:ForgeTestLast     Replay the most recent ForgeTest,
                   ForgeTestMatch or ForgeTestSelection run
:ForgeFmt [args]   Run forge fmt
:ForgeClean [args] Run forge clean
:ForgeInstall [args] Run forge install
:ForgeUpdate [args] Run forge update

CONFIG                                   *forge-config*

require("forge").setup({
  allow_standalone = true,
  split_direction = "top",  -- "top" | "bottom" | "left" | "right"
  match_verbosity = 4,      -- 0-5; appends -v..-vvvvv to
                            -- :ForgeTestMatch and :ForgeTestSelection
                            -- (default 4 = -vvvv, 0 = no flag).
                            -- :Forge test and :ForgeTest are unaffected.
})
